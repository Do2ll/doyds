# 二手物品转售网站 - 设计方案

## 1. 项目概述
- **项目名称**: 二手集市 (SecondHand Mart)
- **项目类型**: 单页面Web应用
- **核心功能**: 用户登录注册、商品浏览、商品发布、二手物品交易平台
- **目标用户**: 想要买卖二手物品的用户

## 2. UI/UX 设计

### 2.1 布局结构
- **导航栏**: 固定顶部，包含Logo、搜索框、导航链接、用户头像/登录按钮
- **Hero区域**: 轮播展示热门商品或促销活动
- **商品分类**: 分类筛选标签
- **商品网格**: 响应式商品卡片展示
- **发布按钮**: 悬浮发布按钮
- **页脚**: 版权信息、联系方式

### 2.2 响应式断点
- 移动端: < 640px (1列)
- 平板: 640px - 1024px (2列)
- 桌面: > 1024px (4列)

### 2.3 配色方案
- **主色**: #10B981 ( emerald-500 绿色 - 代表环保/二手再利用)
- **辅助色**: #F59E0B ( amber-500 橙色 - 代表活力/交易)
- **背景色**: #F9FAFB ( gray-50 )
- **文字色**: #1F2937 ( gray-800 )
- **边框色**: #E5E7EB ( gray-200 )

### 2.4 字体
- 主字体: "Noto Sans SC", sans-serif (支持中文)
- 标题字体: 700 weight
- 正文: 400 weight

## 3. 功能模块

### 3.1 用户认证
- 登录弹窗 (邮箱/用户名 + 密码)
- 注册弹窗 (用户名 + 邮箱 + 密码 + 确认密码)
- 记住我功能
- 退出登录

### 3.2 商品展示
- 商品卡片 (图片、标题、价格、原价、折扣、成色、卖家信息)
- 分类筛选 (电子产品、图书、服装、家居、其他)
- 价格排序 (从低到高、从高到低)
- 商品搜索

### 3.3 商品发布
- 发布表单 (标题、描述、分类、价格、原价、成色、图片上传)
- 表单验证

### 3.4 交互功能
- 加入购物车 / 收藏
- 联系卖家 (模拟弹窗)
- 登录状态切换

## 4. 页面结构
- 单页面应用，通过Tab切换不同视图
- 视图: 首页(商品列表) | 发布商品 | 我的收藏 | 用户中心

## 5. 技术实现
- 纯HTML + CSS + JavaScript
- Tailwind CSS (CDN)
- Font Awesome 图标 (CDN)
- 本地存储模拟数据持久化
