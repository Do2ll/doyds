# DOYDS 二手闲置交易平台

一个简洁的二手物品交易网站，让买卖二手商品变得简单。

## 网站地址

**主站：** https://doyds.xyz

## 功能介绍

### 1. 商品浏览
- 多种分类：手机数码、电脑办公、相机游戏、图书服装等
- 支持搜索和筛选
- 商品排序：最新发布、价格高低

### 2. 用户系统
- 注册账户
- 登录/退出
- 个人中心

### 3. 发布商品
- 上传商品图片
- 填写商品信息（标题、分类、价格、成色、描述）
- 填写联系方式（微信/QQ/电话/邮箱，至少填写一种）

### 4. 购买流程
- 点击商品卡片上的"购买"按钮
- 查看卖家联系方式
- 通过微信/QQ/电话联系卖家
- 直接转账交易

### 5. 收藏功能
- 收藏喜欢的商品
- 查看收藏列表

### 6. 聊天功能
- 与卖家私信沟通

## 技术栈

- **前端：** HTML + Tailwind CSS + JavaScript
- **后端：** 本地存储 (localStorage)
- **图标：** Font Awesome
- **字体：** Google Fonts (Noto Sans SC)
- **托管：** GitHub Pages
